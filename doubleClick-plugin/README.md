### 目的   
为点击事件添加double click判断

### 如何使用
- 在根build.gradle中添加classpath
```Java
classpath "com.mars.infra:doubleClick-plugin:0.0.1"
```
- 在app的build.gradle中apply插件
```java
plugins {
    id 'com.mars.infra.doubleClick.plugin'
}
```
